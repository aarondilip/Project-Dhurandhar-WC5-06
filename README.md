# 🛡️ IB Counter Intelligence Terminal

An immersive, single-file web application that simulates a classified Intelligence Bureau (IB) cyber-intelligence terminal used to decrypt intercepted transmissions.

## ⚠️ Disclaimer

This project is a fictional UI created for educational and creative purposes. It is not affiliated with any real government or intelligence agency.

## 🚀 Features

* Single HTML file (HTML, CSS & JavaScript)
* Military-inspired cyber command interface
* Animated boot sequence and CRT effects
* Real-time packet analysis dashboard
* Custom Caesar-style decryption algorithm
* Interactive mission logs and system status
* Responsive design with keyboard shortcuts
* Web Audio API sound effects

## 🔐 Decryption Logic

1. The **length of the first word** (letters only) is used as the **secret key**.
2. Every alphabetic character in the message is shifted **backward** by the secret key.
3. Letter wrapping is supported (`A → Z`), while case, spaces, numbers, and punctuation are preserved.

## 🛠️ Technologies

* HTML5
* CSS3
* Vanilla JavaScript
* SVG
* Web Audio API

## ▶️ Run

Simply open the `index.html` file in any modern web browser—no installation or dependencies required.
